# Конструктор дакимакуры для InSales

Проект разбит на две части:

- `widget/wt_Konstruktor` — виджет InSales для страницы конструктора
- `backend` — внешний сервис, который создает реальные товары в InSales

Ключевая идея текущей архитектуры:

1. Пользователь собирает дакимакуру в виджете.
2. Виджет отправляет конфигурацию во внешний backend.
3. Backend склеивает preview, создает новый hidden product и новый variant в InSales.
4. В корзину добавляется уже созданный реальный товар.

По запуску и продовому развёртыванию смотрите `DEPLOYMENT.md` и `backend/README.md`.
